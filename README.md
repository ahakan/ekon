
## Installation

### *Installing Prerequisites: CMake*

Please install cmake.

### *Step 1: Clone this project*

```sh
git clone git@github.com:ahakan/ekon.git   
```

### *Step 2.2: Build this project with CMake*
```sh
cd ekon
mkdir build && cd build
cmake .. 
make
```

## Usage

```sh
./ekon     
```
